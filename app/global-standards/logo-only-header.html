Logo Only Header

This page is using the logo-only UTSA global header.

General Guidelines

UTSA global headers are designed to give a consistent branded identity to sites owned and operated by UTSA. Content above the orange line should be considered off-limits to site specific content.

Site specific branding should appear below the orange line to convery a common user experience to all UTSA web users.

Desktop Size, min-width: 1100px

Please note the full UTSA wordmark is used at widths over 1100px on a UTSA blue (#0c2340) background with a maximum width of 320px.


Global top right navigation icons are not included in the logo only global header.

Required Top Level links are not included in the logo only global header.


Small Desktop Size, max-width: 1100px and min-width: 876px

At this breakpoint the UTSA logo switches to the smaller logo-only version with a maximum displayed width of 110px:


Please note, this is the only change to the elements in the global header.

Global top right navigation icons are not included in the logo only global header.

Required Top Level links are not included in the logo only global header.


Tablet Size, max-width: 876px and min-width: 768px

At this breakpoint the UTSA logo has a maximum displayed width of 80px:


Please note, this is the only change to the elements in the global header.

Global top right navigation icons are not included in the logo only global header.

Required Top Level links are not included in the logo only global header.


Breakpoints below 768px

No other changes are made to the global header below 768px. Items to note:

  • The UTSA logo should not be smaller than a width of 80px.
  • The UTSA logo should always be text aligned left.
